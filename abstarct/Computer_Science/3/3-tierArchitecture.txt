1. Structure

- Presentation tier 

 user interface and communiation layer of the application
 interaacts with the application
 
- Application tier

 logic tier & middle tier => heart of the application
 using business logic
 
- Data tier

 called database tier || back-end

2. Benefits of There-tier architecture

- Faster development : each tier can be deveoped simultaneously by different teams

- Improved scalability : independently scaled each of them

- Improved reliability : when outage less impact to others

- Improved security : presentation tier and data tier can't communicate
