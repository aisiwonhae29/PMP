# session

### about

- A session is a way to persist info across multiple requests made by a user to a web app.

### implementation

- Session ID
- Session Storage
- Lifecycle

### usecase

- Authentication: Storing user login state across multiple pages.
- Personalization: Keeping track of user preferences.
- shopping Carts.

- Server-side Storage
