# auditd

### about

- auditd is responsible for powerful auditing framework available in linux that allows administrators to track and monitor security-related events on the system.

### work

1. System auditing

- The primary purpose of 'auditd' is to provide a comprehensive auditing framework for tracking security-relevant events on a linux system.
- This includes monitoring activities such as file access, process execution, system calls, user authentication, etc.

2. Logging and Reporting

- 'auditd' collects audit events generated by the kernel and user-space applications and writes them to log files
